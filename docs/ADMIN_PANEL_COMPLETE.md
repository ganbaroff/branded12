## 🚀 Как использовать

### Управление лидами
1. Перейдите в раздел "Leads"
2. Используйте поиск для быстрого поиска
3. Примените фильтры (статус, score)
4. Выберите лиды через checkbox
5. Используйте bulk actions для массового обновления
6. Измените статус через dropdown в таблице

### Создание блог поста
1. Перейдите в раздел "Blog"
2. Нажмите "New Post"
3. Заполните обязательные поля (slug, title AZ, content AZ)
4. Заполните остальные поля (опционально)
5. Выберите "Publish immediately" или оставьте как черновик
6. Нажмите "Save"

### Редактирование контента
- Все разделы поддерживают inline редактирование
- Изменения сохраняются автоматически при нажатии "Save"

## 📋 Статусы лидов

| Статус | Описание | Когда использовать |
|--------|----------|-------------------|
| **new** | Новый лид | По умолчанию для всех новых лидов |
| **contacted** | Связались | После первого контакта |
| **qualified** | Квалифицирован | Лид готов к продаже |
| **converted** | Клиент | После оплаты |
| **lost** | Потерян | Лид не заинтересован |

## 💡 Best Practices

1. **Обработка лидов**:
   - Сначала обрабатывайте лиды с score 70+
   - Обновляйте статус сразу после контакта
   - Используйте bulk actions для экономии времени

2. **Блог**:
   - Всегда используйте уникальные slug
   - Заполняйте все языки для лучшего SEO
   - Используйте категории для организации

3. **Контент**:
   - Регулярно обновляйте портфолио и кейсы
   - Добавляйте новые отзывы
   - Обновляйте цены и пакеты
