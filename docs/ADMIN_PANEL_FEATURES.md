# 🎛️ Admin Panel - Полный функционал

## ✅ Реализованные функции

### 📊 Dashboard
- Статистика: общее количество лидов, лиды за сегодня, средний score, конверсия
- Распределение по пакетам
- Быстрый доступ ко всем разделам

### 👥 Управление лидами (Leads)
- ✅ **Просмотр всех лидов** с пагинацией (20 на страницу)
- ✅ **Поиск** по имени, email, телефону, компании
- ✅ **Фильтры**:
  - По статусу (new, contacted, qualified, converted, lost)
  - По минимальному score
- ✅ **Управление статусами**:
  - Изменение статуса для каждого лида
  - Bulk update статусов (массовое обновление)
- ✅ **Bulk actions**:
  - Выбор нескольких лидов (checkbox)
  - Массовое изменение статуса
- ✅ **Экспорт в CSV**
- ✅ **Отображение UTM параметров** (источник трафика)
- ✅ **Цветовая индикация score**:
  - Зеленый: 70+ (высокий приоритет)
  - Желтый: 50-69 (средний)
  - Красный: <50 (низкий)

### 📝 Управление блогом (Blog)
- ✅ **Создание новых постов**
- ✅ **Редактирование существующих постов**
- ✅ **Удаление постов**
- ✅ **Поля**:
  - Slug (URL-friendly)
  - Заголовки (AZ/RU/EN)
  - Excerpt (краткое описание)
  - Content (полный текст)
  - Category (категория)
  - Cover Image (обложка)
  - Author (автор)
  - Published Date (дата публикации)
  - Is Published (опубликован/черновик)

### 🎨 Управление портфолио (Portfolio)
- ✅ Редактирование существующих проектов
- ✅ Все поля на 3 языках (AZ/RU/EN)
- ✅ Управление featured проектами
- ✅ Порядок отображения

### 📚 Управление кейсами (Cases)
- ✅ Редактирование case studies
- ✅ Все поля на 3 языках
- ✅ Метрики и результаты
- ✅ Цитаты клиентов

### 💼 Управление услугами (Services)
- ✅ Редактирование услуг
- ✅ Иконки, названия, описания
- ✅ Порядок отображения

### 📦 Управление пакетами (Packages)
- ✅ Редактирование тарифов
- ✅ Цены, features, Stripe links
- ✅ Отметка популярного пакета

### 💬 Управление отзывами (Testimonials)
- ✅ Редактирование отзывов
- ✅ Имена, компании, тексты
- ✅ Фото клиентов

## 🎯 Статусы лидов

| Статус | Описание | Цвет |
|--------|----------|------|
| `new` | Новый лид (по умолчанию) | Серый |
| `contacted` | Связались с клиентом | Желтый |
| `qualified` | Квалифицированный лид | Синий |
| `converted` | Стал клиентом | Зеленый |
| `lost` | Потерянный лид | Красный |

---

## 🔍 Фильтры и поиск

### Поиск
- Ищет по: имени, email, телефону, компании
- Работает в реальном времени (debounced)

### Фильтры
- **Статус**: все / new / contacted / qualified / converted / lost
- **Min Score**: минимальный score лида

---

## 📤 Bulk Actions

1. Выберите лиды через checkbox
2. Нажмите кнопку действия:
   - "Mark Contacted" - отметить как связались
   - "Mark Qualified" - отметить как квалифицированный
   - "Clear" - снять выделение

---

## 🚀 API Endpoints

### Leads
- `GET /api/admin/leads?page=1&limit=20&status=new&search=...&minScore=50`
- `PUT /api/admin/leads/:id/status` - обновить статус
- `POST /api/admin/leads/bulk-status` - массовое обновление
- `DELETE /api/admin/leads/:id` - удалить лид
- `GET /api/admin/export/leads` - экспорт CSV

### Blog
- `GET /api/admin/blog` - все посты
- `POST /api/admin/blog` - создать пост
- `PUT /api/admin/blog/:id` - обновить пост
- `DELETE /api/admin/blog/:id` - удалить пост

## 💡 Советы по использованию

1. **Приоритизация лидов**:
   - Сначала обрабатывайте лиды с score 70+
   - Используйте фильтр по score для быстрого поиска

2. **Статусы**:
   - Обновляйте статус сразу после контакта
   - Используйте "qualified" для лидов, готовых к продаже
   - "converted" - только после оплаты

3. **Блог**:
   - Всегда заполняйте slug (URL-friendly)
   - Используйте категории для организации
   - Сохраняйте как черновик, если не готово

4. **Bulk Actions**:
   - Экономьте время при обработке множества лидов
   - Используйте для массового обновления статусов
